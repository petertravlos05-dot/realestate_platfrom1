# Admin Dashboard - Tab Ραντεβού

## Περιγραφή

Προστέθηκε ένα νέο tab "Ραντεβού" στο admin dashboard που επιτρέπει στους administrators να δουν και να διαχειριστούν όλα τα ραντεβού για όλα τα ακίνητα της πλατφόρμας.

## Χαρακτηριστικά

### 🔍 Φίλτρα Αναζήτησης
- **Αναζήτηση κειμένου**: Αναζήτηση με βάση τίτλο ακινήτου, όνομα αγοραστή/πωλητή, email
- **Κατάσταση**: Φιλτράρισμα με βάση την κατάσταση του ραντεβού (Εκκρεμεί, Εγκεκριμένο, Απορριφθέν, κλπ.)
- **Ακίνητο**: Φιλτράρισμα με βάση συγκεκριμένο ακίνητο
- **Αγοραστής**: Φιλτράρισμα με βάση συγκεκριμένο αγοραστή
- **Πωλητής**: Φιλτράρισμα με βάση συγκεκριμένο πωλητή

### 📊 Στατιστικά
- Σύνολο ραντεβού
- Αριθμός εκκρεμών ραντεβού
- Αριθμός εγκεκριμένων ραντεβού
- Αριθμός απορριφθέντων ραντεβού

### 📋 Πίνακας Ραντεβού
Ο πίνακας εμφανίζει:
- Τίτλος και πόλη ακινήτου
- Όνομα και email αγοραστή
- Όνομα και email πωλητή
- Ημερομηνία ραντεβού
- Ώρα ραντεβού
- Κατάσταση ραντεβού (με χρωματιστά badges)
- Ημερομηνία δημιουργίας

## API Endpoint

### GET /api/admin/appointments

**Παράμετροι:**
- `search`: Αναζήτηση κειμένου
- `status`: Κατάσταση ραντεβού
- `propertyId`: ID ακινήτου
- `buyerId`: ID αγοραστή
- `sellerId`: ID πωλητή

**Απάντηση:**
```json
{
  "appointments": [...],
  "filters": {
    "properties": [...],
    "buyers": [...],
    "sellers": [...]
  }
}
```

## Αρχιτεκτονική

### Frontend
- **Τοποθεσία**: `listings/frontend/src/app/admin/dashboard/page.tsx`
- **State Management**: React useState για τα ραντεβού και τα φίλτρα
- **API Calls**: fetchAppointments() συνάρτηση
- **UI Components**: Tailwind CSS με responsive design

### Backend
- **Τοποθεσία**: `listings/frontend/src/app/api/admin/appointments/route.ts`
- **Database**: Prisma ORM με PostgreSQL
- **Authentication**: NextAuth.js session validation
- **Authorization**: Admin role check

## Χρήση

1. Συνδεθείτε ως admin
2. Πλοηγηθείτε στο `/admin/dashboard`
3. Κάντε κλικ στο tab "Ραντεβού"
4. Χρησιμοποιήστε τα φίλτρα για να βρείτε συγκεκριμένα ραντεβού
5. Παρακολουθήστε τα στατιστικά
6. Κάντε κλικ στο κουμπί "Ανανέωση" για να ανανεώσετε τα δεδομένα

## Ασφάλεια

- Μόνο χρήστες με admin role μπορούν να προσπελάσουν το endpoint
- Session validation με NextAuth.js
- Server-side authorization checks
